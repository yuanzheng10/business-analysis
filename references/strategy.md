# 战略分析框架详情

## <a id="swot"></a>SWOT 分析

**定义**：即态势分析法，20世纪80年代由美国旧金山大学韦里克教授提出。通过评价企业的优势(Strengths)、劣势(Weaknesses)、机会(Opportunities)、威胁(Threats)，对企业进行全面分析。

**分析模板**：
```
【内部因素】
S（优势）：企业内部有哪些突出优势？
  - 资源优势（品牌/技术/资金/人才）
  - 能力优势（管理/创新/运营）
  - 市场优势（份额/客户/渠道）

W（劣势）：企业内部有哪些明显短板？
  - 资源缺口
  - 能力不足
  - 竞争弱点

【外部因素】
O（机会）：外部环境中有哪些可利用的机会？
  - 市场增长机会
  - 政策红利
  - 技术变革机遇
  - 竞争对手的弱点

T（威胁）：外部环境中有哪些潜在威胁？
  - 竞争加剧
  - 替代品威胁
  - 政策监管风险
  - 市场萎缩
```

**关键提示**：SWOT分析后**必须**得出明确结论，不能为了分析而分析。
**局限性**：定性分析，精度有限，带有主观判断。

## <a id="tows"></a>TOWS 分析（基于SWOT的战略制定）

**定义**：TOWS是SWOT的延伸，将分析结论转化为具体战略方案。

**四种战略类型**：
```
SO策略（发展型战略）= 优势 + 机会
→ 发挥内部优势，利用外部机会
→ 积极扩张，抢占市场

WO策略（扭转战略）= 劣势 + 机会
→ 利用外部机会，弥补内部劣势
→ 借势补短，引入外部资源

ST策略（防御型战略）= 优势 + 威胁
→ 利用内部优势，应对外部威胁
→ 以强应变，构建壁垒

WT策略（退却型战略）= 劣势 + 威胁
→ 减少劣势暴露，规避外部威胁
→ 收缩防守，聚焦核心
```

## <a id="pest"></a>PEST 分析

**定义**：外部宏观环境分析工具。

**分析维度**：
```
P（政治/法律）：
  - 政府政策导向
  - 行业监管法规
  - 地方性规定

E（经济）：
  - GDP增速/经济周期
  - 货币政策/利率/汇率
  - 消费结构/通胀水平

S（社会文化）：
  - 人口结构/地理分布
  - 消费观念/生活方式
  - 教育程度/价值取向

T（技术）：
  - 新技术出现速度
  - 产品技术成熟度
  - 技术投资趋势
```

**警示**：PEST针对的是宏观环境，不是每个分析都需要。做不好就成花架子，流于"假大空"。

## <a id="five_forces"></a>波特五力模型

**定义**：Michael Porter于20世纪80年代提出，分析行业基本竞争态势。

**五力分析模板**：
```
① 现有竞争者威胁（核心）：
  - 竞争者集中度与均衡度
  - 行业增长趋势
  - 产品差异化程度
  - 退出壁垒高低

② 新进入者威胁：
  - 进入壁垒（规模经济/资本/渠道/政策）
  - 现有企业的反击能力

③ 替代品威胁：
  - 替代品数量和性价比
  - 客户转换成本
  - 技术发展趋势

④ 供应商讨价还价能力：
  - 供应商集中度
  - 替代供应商可用性
  - 前向整合能力

⑤ 买方讨价还价能力：
  - 买方集中度
  - 购买量占比
  - 产品标准化程度
  - 后向整合能力
```

**局限性**：只考虑竞争关系，未考虑合作共赢的可能性。

## <a id="mckinsey_7s"></a>麦肯锡7S模型

**定义**：企业发展需要全面考虑7个要素的匹配关系。

```
硬件要素（容易改变）：
  Strategy（战略）：企业竞争战略方向
  Structure（结构）：组织架构设置
  System（制度）：运营管理制度和流程

软件要素（难以改变但更重要）：
  Shared Values（共同价值观）：核心文化理念【处于中心位置】
  Style（风格）：管理风格和企业文化
  Staff（人才）：人员素质和构成
  Skill（能力）：组织核心技能

诊断方法：
  对每个S打分(1-5)，找出得分最低的要素 = 管理短板
  检查7S之间的匹配性，不匹配处 = 变革重点
```

## <a id="bcg_matrix"></a>波士顿矩阵（BCG矩阵）

**定义**：BCG提出的投资组合分析方法，用市场增长率×相对市场占有率构成2×2矩阵。

```
四个象限：

明星产品（高增长/高份额）：
→ 战略：积极投资，扩大市场份额

问题产品（高增长/低份额）：
→ 战略：选择性投资，重点扶持有潜力的，放弃其余

现金牛产品（低增长/高份额）：
→ 战略：减少投资，收割现金流，支持其他产品

瘦狗产品（低增长/低份额）：
→ 战略：撤退，剥离，将资源转移到其他产品
```

## <a id="ge_matrix"></a>麦肯锡矩阵（通用矩阵）

**定义**：GE开发，是BCG矩阵的升级版，用多个指标评估两个维度。

```
两个维度：
  X轴 = 竞争实力（内部因素）：
    市场份额/增长率/产品质量/品牌/利润率/技术能力/管理水平

  Y轴 = 市场吸引力（外部因素）：
    市场规模/增长率/盈利性/竞争强度/技术要求/社会政治因素

九宫格战略：
  高吸引力+强竞争力 → 尽量扩大投资，谋求主导地位
  高吸引力+中竞争力 → 市场细分，大力投入
  高吸引力+弱竞争力 → 专门化，考虑并购
  中吸引力+强竞争力 → 选择细分市场，选择性收获
  中+中 → 细分专业化运作
  中+弱 → 专业化，谋求小块市场份额
  低+强 → 收获现金
  低+中 → 减少投资，准备退出
  低+弱 → 集中于可能盈利的业务，或直接退出
```

## <a id="bsc"></a>平衡计分卡（BSC）

**定义**：1992年由Kaplan & Norton提出，从四个维度评价企业绩效。

```
四个维度：
  财务维度：我们在股东眼里表现如何？
    → 净利润、EBITDA率、收入增长、资本回报率

  客户维度：我们在客户眼里表现如何？
    → 满意度、忠诚度、市场份额

  内部流程维度：哪些业务流程最关键？
    → 流程效率、质量指标、创新能力

  学习与成长维度：如何保持创新和持续提高？
    → 人才发展、IT能力、组织学习能力
```

## <a id="value_chain"></a>企业价值链

**定义**：Porter提出，将企业活动分为主要活动和支持性活动，识别价值创造环节。

```
主要活动（直接创造价值）：
  投入性活动 → 生产性活动 → 产出性活动 → 市场与销售 → 服务

支持性活动（辅助主要活动）：
  基础设施与支持
  人力资源与文化管理
  技术的发展
  采购

分析方法：
  对每个活动评估：是竞争优势？还是竞争劣势？
  识别核心竞争力所在的价值链环节
  找出需要改进的薄弱环节
```

## <a id="3c_triangle"></a>3C战略三角

```
三个顶点：
  顾客（Customer）：所有战略的基础
  公司本身（Corporation）：最大化竞争优势
  竞争者（Competitor）：通过差异化在功能领域超越竞争者

核心理念：
  只有将三者整合在同一战略框架内，可持续竞争优势才有可能存在
```

## <a id="space_matrix"></a>SPACE矩阵

**建立步骤**：
```
1. 选择FS（财务优势）、CA（竞争优势）、ES（环境稳定性）、IS（产业优势）各3-5个指标
2. FS和IS评分：+1（最差）到+6（最好）
   CA和ES评分：-1（最好）到-6（最差）
3. 计算各轴平均分，在坐标轴标出
4. X轴 = IS均值 + CA均值，Y轴 = FS均值 + ES均值
5. 从原点到交叉点画箭头，指向象限即为战略类型

四种战略类型：
  进取型（右上）：财务强+环境稳定，积极进攻
  竞争型（右下）：产业吸引力高但财务弱，提升竞争力
  保守型（左上）：财务强但市场不稳，保守稳健
  防御型（左下）：财务弱+环境不稳，收缩防御
```

## <a id="strategy_clock"></a>战略钟模型

```
8种战略：
  1. 低价低值（集中成本领先）
  2. 低价格（成本领先）：容易被模仿
  3. 混合战略：高品质+低价格，难以被模仿
  4. 差异化：高品质+稍高价格
  5. 集中差异化：极高价格+极高品质
  6. 高价撇脂：不考虑价值，垄断行业可用
  7-8. 失败战略：高价低值（不可持续）
```

## <a id="blm"></a>BLM业务领先模型

```
框架结构（从左到右）：
  战略侧：
    市场洞察 → 战略意图 ↕ 创新焦点 → 业务设计
  执行侧：
    氛围文化 ↕ 正式组织 ↕ 人才 → 关键任务依赖关系
  
  上方：领导力（贯穿始终）
  下方：价值观（底层基础）
  右侧：市场结果（差距=业绩差距+机会差距）

使用方法：
  从"差距"出发，倒推战略和执行上的问题所在
```

## <a id="tech_maturity"></a>技术成熟度曲线

```
五个阶段：
  科技诞生期：新技术出现，少数创业者尝试
  泡沫巅峰期：媒体过度报道，企业获得高估值
  泡沫破裂低谷期：技术问题暴露，泡沫破灭
  稳步爬升期：客观认识技术，摸索商业模式
  规模应用期：熟练掌握，大规模商业化

决策意义：
  巅峰期 → 谨慎投资，不追风口
  低谷期 → 可能是最佳投资时机
  爬升期 → 稳步布局
```

## <a id="s_curve"></a>市场成长S形曲线

```
四个阶段及工作重点：
  导入期：寻找市场突破口，培养领先试用者，形成示范效应
  成长期：实现规模化销售，提高市场占有率
  成熟期：维护产品，提高客户离网成本，防止流失
  衰退期：促使老客户向新产品转移，实现交叉销售

关键点：
  破局点（导入→成长）：找到可复制的增长模式
  拐点（成长→成熟）：市场开始饱和，竞争加剧
  极限点（成熟→衰退）：必须提前布局下一条S曲线
```
