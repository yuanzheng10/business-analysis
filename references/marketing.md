# 营销分析框架详情

## <a id="4p"></a>4P 理论

**定义**：营销学最经典的组合理论，企业一切营销动作围绕四个要素展开。

```
P1 产品（Product）：
  - 产品功能/性能/设计
  - 产品线宽度与深度
  - 品牌/包装/服务
  - 新产品开发策略

P2 价格（Price）：
  - 定价方法（成本导向/竞争导向/价值导向）
  - 折扣策略/付款条件
  - 价格歧视（不同客群不同价）
  - 影响定价三要素：需求、成本、竞争

P3 渠道（Place）：
  - 直销 vs 分销
  - 渠道层级与密度
  - 线上线下渠道整合
  - 渠道激励政策

P4 促销（Promotion）：
  - 广告（品牌认知）
  - 销售促进（短期刺激）
  - 公关（口碑/媒体）
  - 人员销售（B2B重要）
```

**4P诊断方法**：
对每个P打分（1-5），得分最低的 = 当前营销主要短板

## <a id="4c_4r"></a>4C-4R 理论

```
4C（以顾客为中心）：
  顾客需求（Consumer's Needs）：不是卖产品，而是满足需求
  客户支付成本（Cost）：顾客购买的总成本（含时间/精力）
  购买便利性（Convenience）：让客户买得更方便
  客户沟通（Communication）：双向沟通而非单向推销

4R（关系营销）：
  关联（Relevancy）：与顾客建立相关性
  节省（Retrenchment）：帮助顾客节省时间/金钱
  关系（Relationship）：建立长期关系
  报酬（Rewards）：给予顾客回报激励
```

## <a id="stp"></a>STP 理论

**三步流程**：
```
S（市场细分 Segmentation）：
  细分维度：
    地理变量：地区/城市规模/气候
    人口变量：年龄/性别/收入/职业/教育
    心理变量：生活方式/价值观/个性
    行为变量：使用频率/品牌忠诚度/购买动机

T（目标市场 Targeting）：
  评估标准：
    市场规模与增长潜力
    竞争强度
    企业资源匹配度
  
  目标市场策略：
    无差异营销（全市场）
    差异化营销（多个细分）
    集中营销（专注一个细分）

P（市场定位 Positioning）：
  定位主张设计：
    "对于[目标客户]，[产品名]是[产品类别]中
     能够[核心利益]的产品，不同于[竞争对手]，
     我们的[差异化依据]"
```

## <a id="aida"></a>AIDA / AIDMA 模型

```
AIDA（基础版）：
  Attention（注意）→ Interest（兴趣）→ Desire（欲望）→ Action（行动）

AIDMA（记忆增强版）：
  Attention → Interest → Desire → Memory（记忆）→ Action

各阶段营销重点：
  注意阶段：广告投放/社交媒体/KOL/SEO
  兴趣阶段：内容营销/产品演示/体验活动
  欲望阶段：案例/口碑/深度内容/个性化推荐
  记忆阶段：品牌符号/口号/重复曝光
  行动阶段：促销/限时/简化购买流程
```

## <a id="usp"></a>USP 独特销售主张

```
三要素：
  利益承诺：告诉消费者能获得什么具体利益
  独特性：这个利益是竞争对手无法提供的
  强而有力：聚焦在一个点上，而非面面俱到

提炼流程：
  1. 列出产品所有特性
  2. 将特性转化为客户利益（特性→利益转化）
  3. 识别竞争对手无法轻易复制的利益点
  4. 压缩为一句话：[目标客户]用了[产品]，能[具体利益]
```

## <a id="ansoff"></a>安索夫矩阵

```
四种增长策略（现有/新产品 × 现有/新市场）：

市场渗透（现有产品×现有市场）：
  核心问题：如何让现有客户买得更多？
  策略：降价/捆绑/促销/提高使用频率/争夺竞争对手客户

产品开发（新产品×现有市场）：
  核心问题：给现有客户卖什么新东西？
  策略：产品延伸/新功能/新品类/捆绑销售

市场开发（现有产品×新市场）：
  核心问题：同样的产品去哪里卖？
  策略：地理扩张/新渠道/新客群/国际化

多元化（新产品×新市场）：
  核心问题：全面进入陌生领域（风险最高）
  策略：相关多元化（协同效应）/非相关多元化
```

## <a id="customer_pyramid"></a>顾客金字塔模型

```
四层结构（从上到下，价值递减）：
  铂层顾客：最高利润贡献，约占5-10%
    → 策略：专属服务/个性化体验/优先资源
  
  金层顾客：重要利润贡献，约占15-20%
    → 策略：提升至铂层/强化关系/交叉销售
  
  铁层顾客：基础利润贡献，约占30-40%
    → 策略：标准化服务/自助服务/效率优先
  
  铅层顾客：成本高于利润（负利润客户）
    → 策略：提价/减少服务成本/考虑放弃

应用方法：
  1. 计算每个客户的利润贡献（CLV）
  2. 按利润贡献分层
  3. 差异化分配服务资源
```

## <a id="brand_health"></a>品牌健康度诊断模型

```
五维度评估：
  认知度：目标客群中有多少人知道这个品牌？
  美誉度：知道的人中有多少人评价正面？
  使用率：使用过的人占知道的人的比例
  购买性：愿意再次购买的比例
  推荐率：会向他人推荐的比例（NPS）

四种品牌类型：
  健康高认知品牌：五维均衡且得分高（理想状态）
  传播缺乏品牌：认知度低但其他维度尚可（加大传播）
  健康低认知品牌：美誉度高但认知度低（扩大触达）
  虚名品牌：认知度高但购买/推荐低（改善体验）

诊断工具：将五维画成漏斗图，
  漏斗越均匀 = 品牌越健康
  某处突然收窄 = 该维度是瓶颈
```

## <a id="life_cycle"></a>生命周期理论

### 客户生命周期
```
阶段：观望期→体验期→初入期→忠诚期→异动期→预离期→离网期

各阶段顾虑与营销重点：
  观望期：产品功能顾虑 → 免费试用/演示
  体验期：换号/换机/积分问题 → 引导激活/服务专席
  初入期：网络/手机质量问题 → 快速响应/优化体验
  忠诚期：营销骚扰/优惠服务 → 减少打扰/精准推送
  异动期：产品价格/用户服务 → 挽留方案/VIP升级
  预离期：业务优惠 → 精准报价/换约激励
  离网期：离网手续 → 简化流程（争取未来回归）
```

### 产品生命周期
```
导入期：找市场突破口，培养领先试用者，形成示范效应
成长期：规模化销售，提高市场占有率
成熟期：维护产品，提高客户离网成本（如API集成）
衰退期：促使老客户向新产品迁移，交叉销售

关键动作：在衰退期之前就要开始孵化下一代产品（S曲线接力）
```
